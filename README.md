## 📂 Directory Structure

| 폴더명 | 설명 |
| --- | --- |
| `백준 /` | 백준 온라인 저지(BOJ) 풀이 소스 코드 (난이도별 관리) |
| `프로그래머스 /` | 프로그래머스(Programmers) 레벨별 문제 풀이 |
| `LeetCode /` | 리트코드 글로벌 문제 풀이 및 인터뷰 대비 |
| `src /` | **Samsung Expert Academy (SWEA) 문제 풀이** |
| `┗ src/algorithm/` | 핵심 알고리즘 문제 해결 소스 코드 (BFS/DFS, DP, 최적화 등) |
| `┗ src/datastructure/` | 구조적 접근을 위한 자료구조 소스 코드 (Trie, Tree, Graph 등) |
| `scripts /` | 런타임/메모리 포맷팅 및 아카이빙 관리 자동화 스크립트 |
